◆ Obj_Error_Code

enum Obj_Error_Code

Enum for defining the errors that we can encounter OBJ_ERR_SUCCESS = action completed successfully OBJ_ERR_NO_INDEX = there is no object in the objects array that has the given index OBJ_ERR_NO_SUB_INDEX = there is no subindex object of an index object with given subindex number OBJ_ERR_WRITEONLY = tried to read writeonly object OBJ_ERR_READONLY = tried to write to a readonly object OBJ_ERR_NULL_PARAM = control object which is used to reach the actual object is null
